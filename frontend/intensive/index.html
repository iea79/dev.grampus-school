Что вы получите?

По опыту - навыки работы с js один из самых востребованных и дорогих при работе верстальщика.

Что будет на интенсиве:

  • Отфильтруем только необходиме знания для работы верстальщика на реальных проектах
  • Разберемся с основами (функции, переменные, операторы и др.)
  • Разберемся с тем что такое база данных, из чего состоит, как с ней работать и зачем все это знать верстальщику
  • Научимся писать красивый, структурированный и понятный код
  • Научимся работать с данными которые программисты передают вам с сервера и формировать из них готовый HTML
  • Научимся создавать пагинацию и подгружать(а не просто скрывать/показывать) дополнительные блоки на странице
  • Научимся работать с одностраничными приложениями где данные на страницы меняются без перезагрузки страницы
  • Значительно повысим понимание разработки сатов в целом (то чего не хватает многим верстальщикам на фрилансе)
  • и это далеко не все...

В итоге вы получаете значительное конкурентное преимущество перед многими html-верстальщиками и как результат больше зарабатываете. И приятным бонусом - углубленное понимание процесса разработки сайта и взаимодействия с программистами.

В программе:

  • День 1
    "Вспоминаем основы js(jQuery)"
    • переменные
    • операторы
    • функции, аргументы, callback
    • циклы
    • массивы
    • консоль
    • вывод результатов в html
  • День 2
    "Принципы работы с БД"
    • структура
    • наполнение
    • получаем данные в формате JSON
    • разбираем JSON
    • браузерные инструменты
  • День 3 и 4
    "Объединяем воедино"
    • AJAX
    • отправка запросов на сервер
    • получение и разбор ответов от сервера
    • вывод в текущую верстку
    • обновление без перезагрузки
    • простая пагинация
    • выпадающий список при поиске
    • фильтрация
